About Time Wheel

Time Wheel is a modern web application for managing your time and tasks efficiently.

Features

  • User authentication with secure password management
  • PIN-based quick login for trusted devices
  • Admin panel for user management
  • Real-time time tracking
  • Responsive design

Technology Stack

  • Backend: Go 1.24
  • Database: SQLite3
  • Frontend: HTML5, CSS3, Vanilla JavaScript
  • Security: bcrypt password hashing, secure session management

Version

Version 1.0.0
